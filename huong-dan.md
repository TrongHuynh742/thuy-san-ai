# 🦐 Cổng Thông Tin Số Hóa Nuôi Trồng Thủy Sản AI
Dự án website tự động cập nhật tin tức và tiêu chuẩn quản trị (ASC, Organic, Fair Trade) trong ngành nuôi trồng thủy sản bằng Trí tuệ Nhân tạo (AI).

## Tính năng nổi bật
* **Giao diện hiện đại:** Được thiết kế tối ưu cho việc đọc trên máy tính và điện thoại.
* **Tự động hóa 100%:** Sử dụng Python và GitHub Actions để tự động chạy lúc 6h sáng mỗi ngày.
* **Tích hợp AI:** Kết nối với Google Gemini API để tự động đọc báo, chọn lọc tin tức quan trọng và tóm tắt thành các gạch đầu dòng ngắn gọn.
